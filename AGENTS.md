# Denuvo Slides Template

A Slidev presentation about reverse engineering Denuvo in Hogwarts Legacy, presented at Navaja Negra 2025.

## Tech Stack

- **Framework**: Slidev v52
- **Language**: Vue 3 / Markdown
- **Package Manager**: npm
- **Build Output**: `dist`

## Available Skills

| Skill | Description |
|-------|-------------|
| `build-and-deploy` | Build and deploy the presentation to Vercel or Netlify |

## Quick Start

```bash
npm install
npm run dev
```

## Customization

- Edit `slides.md` to modify the presentation content
- Slidev uses Markdown with Vue components for slides
- See [Slidev documentation](https://sli.dev) for more features
